# Programmering med Python

## **Programmering med Python**

Denna guide lägger fokus på tydliga kodexempel \- förklarande texter hålls korta. Syftet är att guiden ska kunna användas praktiskt som en referens eller "kokbok" när man programmerar med Python.

### **Guidens fyra delar**

* [**Grunder**](https://python.ostrawebb.se/grunder.html) \- här tas grunderna i Python upp: variabler, datatyper, if-satser, loopar, funktioner etc. Här finns också exempel från modulerna datetime, csv och random.  
* [**Fortsättning**](https://python.ostrawebb.se/fortsattning.html) \- denna del bygger vidare med exempel på hur man skapar egna moduler, hanterar JSON-data, objektorienterad programmering, list comprehensions och mer avancerad list-bearbetning. Här tas också datatypen set upp och hur man arbetar med mängder.  
* [**Tillämpning**](https://python.ostrawebb.se/tillampning.html) \- denna del visar hur man kan använda programmeringen i praktiska tillämpningar: bildbearbetning, skapa webbappar, dataanalys, visualisering, web scraping etc.  
* [**Verktyg**](https://python.ostrawebb.se/verktyg.html) \- denna del tar upp de verktyg som används inom programmering, versionshantering med Git och GitHub, virtuella maskiner i Github Codespaces, vanliga kommandon i Linux-terminalen etc.

### **Färgkoder för att visa nivån**

Många delar av guiden är färgkodade för att visa svårighetsnivån på innehållet:

⏺ **Grönt** anger grundläggande kunskap som man bör vara väl förtrogen med. Det här är sådant som är bra att lära sig utantill för att kunna programmera effektivt med Python.

⏹ **Gult** betyder att exemplet är lite mer avancerat, eller att det som exemplet tar upp inte är lika centralt. Man bör förstå dessa exempel, men det är inget som man behöver kunna utantill.

◆ **Rött** betyder att det som tas upp är ännu mer avancerat, eller är något som inte är så vanligt förekommande inom Python på nybörjarnivå.
